# Nine-Week Capstone Learning Program

## Purpose

This program expands the intensive four-week stream into a paced 63-block sequence with spaced retrieval, deliberate catch-up, cumulative integration and mock assessment. It is an original learning pathway, not a substitute for authorised standards, legislation, regulator guidance or RTO procedures.

## Sequence rules

- Every scheduled block has one dedicated eight-beat module and one knowledge-base note.
- New concepts are followed by retrieval, varied application and cumulative integration.
- Safety-critical procedures remain `review-required` and `reference_check_required` until checked by a qualified reviewer against current authorised sources.
- Exact clauses, values, limits and official assessment claims are not supplied from memory.
