# Six-Week Day 15 - Load Identification and Maximum-Demand Workflow

## Purpose

Build a traceable load model before using an authorised maximum-demand method. Separate connected load, assessed contribution and design margin, and keep phase and operating-case assumptions visible.

## Core workflow

**L-O-A-D-S:** List the scope → Obtain the governing method → Align units and assumptions → Determine group and phase contributions → Stress-test and state the bounded result.

## Practical application

Create a fictional load register, compare three operating cases and revise the model when one control assumption changes. Do not invent missing rating bases, phase data or demand allowances.

## Assessment relevance

The learner must classify loads, select the authorised source family, preserve units and assumptions, test phase and operating cases, and state an evidence-bounded design input.
