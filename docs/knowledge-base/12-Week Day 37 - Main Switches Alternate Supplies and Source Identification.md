# 12-Week Day 37 - Main Switches Alternate Supplies and Source Identification

## Purpose

Builds complete source inventories and operating-state maps before any main-switch, source-identification or isolation claim is made.

## Core concepts

Main switch; alternate supply; stored energy; backfeed; source inventory; operating-state matrix; identification evidence.

## Workflow

**S-O-U-R-C-E** surveys the boundary, outlines states, uncovers every energisation path, relates switching functions to sources, checks identification evidence and expresses unresolved risks.

## Practical application and assessment relevance

Learners create source inventories, state matrices and switch-function maps for original scenarios, then explain what reopens when a source is added. This supports Capstone preparation but is not an official assessment or technical approval.

## Safety boundary

Written source-mapping only. Never infer isolation from one open device, absent indication or a familiar arrangement. No field procedure is authorised.
