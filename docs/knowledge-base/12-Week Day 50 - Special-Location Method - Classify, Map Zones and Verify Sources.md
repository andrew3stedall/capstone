# 12-Week Day 50 - Special-Location Method - Classify, Map Zones and Verify Sources

Day 50 extends [[12-Week Day 49 - Week 7 Installation Planning Exercise]] by adding a reusable method for classifying special locations before selecting requirements.

## Learning module

- [Day 50 — Special-Location Method: Classify, Map Zones and Verify Sources](../learning-plans/12-week/modules/day-50-special-location-method-classify-map-zones-and-verify-sources.md)

## Core workflow

Use **Z-O-N-E-S**: zero in on facts, outline exposure, name candidate rule sets, establish applicability and state a bounded conclusion.

## Related concepts

- [[Wiring Rules and Design]]
- [[Safety and Electrical Risk]]
- [[Inspection Testing and Verification]]
- [[12-Week Day 44 - Environmental Influences, Segregation and Support Concepts]]

## Practical application

Learners create an original exposure map, identify interacting rule sources, grade evidence and reopen the analysis after a movable or fixed exposure source changes.

## Assessment relevance

A strong response classifies from facts rather than room labels, separates learner maps from official zones, verifies source applicability and stops when material evidence is missing.

## Safety boundary

This note authorises no site classification, measurement, opening, switching, isolation, testing, installation, alteration, energisation, certification or verification.
