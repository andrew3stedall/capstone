# Six-Week Day 36 - Verification Purpose Evidence and Visual Inspection
## Purpose
Separate visual-inspection evidence, authorised test evidence and unresolved evidence before forming a bounded verification record.
## Core workflow
**V-I-S-U-A-L:** Verify scope → Identify authorised questions → Separate facts → Unresolved items become requests → Assign traceability → Limit the conclusion.

## Practical application
Build a fictional visual-inspection record that identifies what inspection establishes and what still requires test evidence or escalation.
## Assessment relevance
Tests evidence classification, scope control, traceability and refusal to overclaim certification.
