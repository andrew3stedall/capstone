# Six-Week Day 31 - Fixed Appliances Local Isolation and Connection Decisions

## Purpose

Build a traceable fixed-appliance decision by separating functional control, protection, connection and isolation, mapping every source state and reopening dependent conclusions when conditions change.

## Core workflow

**I-S-O-L-A-T-E:** Identify equipment and task boundary → Survey every source and state → Observe controls, protection and connection data → Locate candidates → Access current authorised and manufacturer evidence → Test dependencies against changed states → Express grades, limits and bounded action.

## Evidence and claim model

- Evidence grades: **recalled**, **located**, **supported**, **transferred**, **unresolved**.
- Claim grades: **memory claim**, **provisional interpretation**, **supported study conclusion**, **authorised technical determination**.
- Appliance decision ledger: boundary → sources and states → controls → protection → candidate connection → candidate isolation → manufacturer evidence → grades → dependencies → reopening triggers → bounded action.
- A label, nearby switch, protective device, plug, normal stop response or remembered arrangement is not proof of a complete isolation boundary.

## Practical application

Complete appliance decision ledgers for fictional water-heating, commercial-cooking and ventilation packs. Separate observations from inferred functions, grade evidence and claims, compare candidate connection arrangements, define the boundary that requires proof, perform one changed-condition transfer and state a bounded action without carrying out an isolation procedure.

## Assessment relevance

Tests observable reasoning across task-boundary definition, source-state completeness, control/protection/connection/isolation distinctions, evidence grading, dependency reopening, changed-condition transfer and bounded reporting. The 12-point rubric and critical-error gates are original educational tools, not an official RTO pass mark.

## Reopening triggers

Reopen the conclusion when the appliance model, manufacturer instructions, task boundary, source arrangement, control mode, automatic-restart behaviour, stored-energy condition, connection details, protective arrangement, access, jurisdiction or source currency changes or conflicts with later evidence.
