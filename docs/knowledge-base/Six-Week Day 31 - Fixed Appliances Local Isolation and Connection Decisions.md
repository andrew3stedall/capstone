# Six-Week Day 31 - Fixed Appliances Local Isolation and Connection Decisions
## Purpose
Separate operational control, connection and isolation decisions by mapping equipment boundaries and every relevant energy source.
## Core workflow
**I-S-O-L-A-T-E:** Identify boundary → Survey sources → Observe controls → Locate candidates → Access authorised evidence → Test changed states → Express bounded decision.

## Practical application
Compare appliance source maps, controls, connection options, isolation candidates and missing evidence without performing an isolation procedure.
## Assessment relevance
Tests whether the learner can distinguish control, protection and isolation and can reopen reasoning after remote or stored energy is introduced.
