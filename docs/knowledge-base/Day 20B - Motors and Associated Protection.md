## Core model

Use **M-O-T-O-R**:

- **Machine and mission**
- **Operating cases**
- **Thermal and fault protection**
- **Operating controls and isolation**
- **Records and review**

The central idea is that a motor installation is a coordinated electromechanical system. The motor, driven machine, cable, starter, protective functions, controls, restart logic and isolation boundary must be reviewed together.

## Practical application

Build an operating-case register and function map for a fictional motor system. Separate normal start, running, overload, short circuit, stall, abnormal supply, remote control, automatic restart and maintenance states. Stop before selecting settings or describing physical work when evidence is incomplete.

## Assessment relevance

A strong response distinguishes protection functions, includes the driven machine, checks coordination and recognises that a stopped motor is neither necessarily isolated nor guaranteed to remain stopped.
