# Knowledge Library

## Status

Milestone 3 engineering complete; technical review outstanding.

## Purpose

The knowledge library turns the capability map into searchable, connected learning articles. It uses original explanations and scenarios and sends learners to authorised sources for exact requirements.

## Implemented capabilities

- structured Zod-validated sections and metadata;
- searchable titles, objectives, tags and article text;
- generated tables of contents and backlinks;
- linked plain-language glossary;
- source edition and checked-date notices;
- explicit reviewer and content-state metadata.
