# Six-Week Day 40 - Rest Final Catch-Up and Readiness Triage

## Purpose

Recovery and readiness block using **R-E-A-D-Y**: retrieve before rereading; examine evidence and earliest causes; allocate no more than three repairs; demonstrate repair and transfer; yield to stop conditions and state a bounded decision.

## Evidence and claim model

- Evidence grades: **stated**, **indicated**, **corroborated**, **transferred**, **unresolved**.
- Claim grades: **assumption**, **provisional educational conclusion**, **supported educational conclusion**, **authorised technical determination**.
- Readiness-evidence ledger: domain/task → error → earliest cause → consequence → prerequisite reach → evidence source/date → evidence grade → closed-note result → changed-condition result → fatigue/uncertainty signal → claim grade → decision → reopening trigger.
- A copied correction, familiar item, confidence statement or long study session cannot independently establish readiness.

## Practical application

Use a strict 30-minute recovery block: retrieve across six domains, grade evidence, select no more than three repairs, correct the earliest causal errors, complete changed-condition transfer, then record **ready to attempt**, **ready with named limits**, **defer and repair**, or **escalate for qualified review**. Record every deferral and reopening trigger rather than treating unfinished work as complete.

## Assessment relevance

The learner must demonstrate observable performance in:

- independent closed-note retrieval;
- causal-error selection rather than easy-task selection;
- traceable repair using identified learning sources;
- changed-condition transfer;
- fatigue, uncertainty and stop-condition judgement;
- bounded readiness reporting; and
- explicit separation of paper-based readiness from practical competence or technical approval.

The six-category 12-point rubric and critical-error gates are original educational tools, not an official RTO pass mark or qualification decision.

## Reopening triggers

Reopen affected rows when task scope, source revision, terminology, diagram, calculation conditions, equipment identity, supply state, assessment format, available time, prompting level, fatigue state or responsible reviewer changes.
