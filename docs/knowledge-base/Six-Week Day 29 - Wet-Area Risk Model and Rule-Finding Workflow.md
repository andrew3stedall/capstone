# Six-Week Day 29 - Wet-Area Risk Model and Rule-Finding Workflow
## Purpose
Map water, contact and geometry risks before locating current authorised special-location requirements.
## Core workflow
**W-A-T-E-R-S:** Write scenario → Anchor geometry → Trace exposure → Establish classification → Retrieve requirements → State bounded result.

## Practical application
Create a scenario sheet that separates water source, geometry, exposure pathway, authorised query, missing evidence and bounded decision.
## Assessment relevance
Tests special-location risk reasoning without invented zone dimensions, equipment ratings or compliance claims.
