# Earthing Bonding and MEN

This map of content connects the learning sequence for protective earthing, equipotential bonding and the multiple earthed neutral system. It is an index and conceptual guide, not a substitute for current authorised standards.

## Learning sequence

1. [[Day 06A - Earthing Terminology and Component Roles]] — identify each component and state its normal and fault role.
2. [[Day 06B - MEN Fault-Current Path]] — trace an active-to-enclosure fault through the protective metallic path and back to the source.
3. [[Day 06C - Earthing and MEN Fault Scenarios]] — diagnose missing, misplaced or ineffective protective connections.
4. [[Day 07 - Week 1 Consolidation and Competency Check]] — retrieve and apply the complete Week 1 model.

## Practical application

Use this map when a learner must:

- label an earthing-system diagram;
- trace a fault-current path;
- distinguish neutral current from protective-conductor current;
- classify exposed and extraneous conductive parts;
- identify a missing or misplaced connection;
- explain why an electrode, MEN connection, protective earthing conductor and bonding conductor are not interchangeable;
- decide which exact requirement must be verified before acting.

## Assessment relevance

Capstone responses should explain the complete relationship between components rather than provide isolated definitions. Credit-worthy reasoning normally includes:

1. component identification;
2. normal operating role;
3. fault-condition role;
4. connection to adjacent components;
5. expected protective outcome;
6. traceable source verification for exact requirements.
