# Six-Week Day 35 - Week 5 Integrated Installation Inspection

## Purpose

Integrate Week 5 location, equipment, operating-state and source evidence into a traceable, bounded paper-based inspection report without upgrading assumptions into technical determinations.

## Core workflow

**I-N-T-E-G-R-A-T-E:** Identify scope and sources → Note observations → Triage triggers → Examine interactions and states → Gather authorised evidence → Record grades and dependencies → Apply stop boundaries → Test changed conditions → End with a bounded report.

## Evidence and claim model

- Evidence grades: **stated**, **indicated**, **corroborated**, **transferred**, **unresolved**.
- Claim grades: **assumption**, **provisional educational conclusion**, **supported educational conclusion**, **authorised technical determination**.
- Integrated ledger: scope item → observation → trigger → state/source path → interaction → evidence grade → claim grade → dependency → request → reopening trigger → stop/escalation → bounded conclusion.
- Labels, drawings, photographs and normal operating responses may contribute evidence but do not independently prove compliance, defect status, isolation, suitability or safety.

## Practical application

Produce a fictional integrated report with a source-state inventory, graded observations and claims, candidate concerns, interaction checks, evidence requests, dependencies, reopening triggers, stop conditions, changed-source and changed-location transfer, and a bounded conclusion.

## Assessment relevance

Tests observable performance in cumulative inspection reasoning, evidence grading, interaction control, source and state mapping, selective change propagation and safe reporting boundaries. The six-category 12-point rubric and critical-error gates are original educational tools, not an official RTO pass mark.

## Reopening triggers

Reopen affected ledger rows when scope, task boundary, location geometry or classification, equipment information, motor duty, operating state, control mode, source type or availability, transfer behaviour, stored energy, drawing revision, authorised reference, jurisdiction, evidence currency, competence or authority changes or conflicts with later evidence.
