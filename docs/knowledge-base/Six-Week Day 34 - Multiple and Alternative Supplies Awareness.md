# Six-Week Day 34 - Multiple and Alternative Supplies Awareness
## Purpose
Map all energy sources and operating states before accepting an energisation, transfer or isolation claim.
## Core workflow
**S-O-U-R-C-E-S:** Scan sources → Outline states → Understand paths → Record evidence → Check boundary → Escalate gaps → State bounded result.

- [[Six-Week Day 33 - Rest Retrieval and Scenario Triage]]
- [[Isolation Boundaries]]

## Practical application
Build a source-state matrix and test a proposed isolation boundary against every stated operating state.
## Assessment relevance
Tests source inventory, changed-state reasoning, evidence control and refusal to overclaim safety.
