# Six-Week Day 34 - Multiple and Alternative Supplies Awareness

## Purpose

Map every stated, indicated and unresolved energy source across material operating states before accepting an energisation, transfer, inspection or isolation claim.

## Core workflow

**S-O-U-R-C-E-S:** Scan every source → Outline operating states → Understand energisation paths → Record evidence and dependencies → Check the boundary → Escalate unresolved interactions → State a bounded conclusion.

## Evidence and claim model

- Evidence grades: **stated**, **indicated**, **corroborated**, **transferred**, **unresolved**.
- Claim grades: **assumption**, **provisional educational conclusion**, **supported educational conclusion**, **authorised technical determination**.
- Source-state evidence ledger: source candidate → operating state → candidate path → affected section → evidence grade → transfer or control dependency → boundary implication → reopening trigger → bounded conclusion.
- A label, drawing, indicator, switch position or normal stop response may contribute evidence but does not independently prove source absence, transfer behaviour, isolation or safety.

## Practical application

Build a source-state evidence ledger for four fictional installations, test each proposed boundary against every material operating state, apply one changed-state and one changed-boundary challenge, and separate supported conclusions from unresolved items requiring authorised verification.

## Assessment relevance

Tests observable performance in source inventory, state separation, path tracing, evidence grading, dependency control, change propagation, boundary discipline and refusal to overclaim safety. The six-category 12-point rubric and critical-error gates are original educational tools, not an official RTO pass mark.

## Reopening triggers

Reopen affected rows when source type, availability, connection, operating state, transfer behaviour, interlocking, control mode, remote command, automatic restoration, stored energy, auxiliary supply, task boundary, drawing revision, product information, jurisdiction, evidence currency, competence or authority changes or conflicts with later evidence.
