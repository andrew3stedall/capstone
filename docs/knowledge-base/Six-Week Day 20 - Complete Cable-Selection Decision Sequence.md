# Six-Week Day 20 - Complete Cable-Selection Decision Sequence
## Purpose
Integrate load, protection, conductor capacity, route conditions, voltage drop, terminals and equipment constraints into one traceable decision sequence.
## Core workflow
**S-E-L-E-C-T-I-O-N:** Scope → Establish load → Link roles → Evaluate conditions → Check protection → Test voltage drop → Inspect constraints → Observe interactions → Note bounded conclusion.
