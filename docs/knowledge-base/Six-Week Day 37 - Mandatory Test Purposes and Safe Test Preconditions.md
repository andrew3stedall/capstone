# Six-Week Day 37 - Mandatory Test Purposes and Safe Test Preconditions
## Purpose
Match a verification question to its evidence purpose and identify every precondition that must be resolved before an authorised method begins.
## Core workflow
**P-R-E-C-H-E-C-K:** Purpose → Reference → Establish scope/sources → Confirm authority → Hazards → Equipment → Circuit state → Keep the stop decision.
