# Block 20 - Cumulative Fault-Path Exercise

## Core recall
A sound fault-path conclusion separates the normal path, initiating fault, protective path and possible body-contact path. Every path segment needs an evidence label, and a plausible loop is not proof of protective performance.

## Retrieval prompts
- What facts and operating state are fixed by the scenario?
- Which path segments are stated, shown, inferred, assumed or unknown?
- Where does the proposed loop return to the source?
- Which contradiction or evidence gap creates a stop condition?
