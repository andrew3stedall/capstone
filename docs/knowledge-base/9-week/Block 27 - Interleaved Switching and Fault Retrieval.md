# Block 27 — Interleaved Switching and Fault Retrieval

## Core idea
Mixed practice is useful only when the learner first discriminates which concept applies: control purpose, source state, fault path or evidence sufficiency.

## Retrieval prompts
- Which facts describe control purpose?
- Which facts change the source-state map?
- What evidence is needed to trace a complete fault loop?
- What stop condition prevents overclaiming?

## Evidence habit
Label facts, inferences, assumptions and unknowns before stating a conclusion. Calibrate confidence to evidence quality, not familiarity with the topic.

## Safety boundary
This note does not authorise field switching, isolation, testing or fault investigation. Current authorised procedures and qualified supervision remain necessary.
