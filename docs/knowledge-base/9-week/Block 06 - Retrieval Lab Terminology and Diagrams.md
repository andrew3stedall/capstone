# Block 06 - Retrieval Lab Terminology and Diagrams

## Core idea

Closed-book retrieval reveals usable knowledge. Check confidence against accuracy, separate diagram interpretation from real-world evidence, and convert each error into a specific delayed retrieval task.

## Retrieval prompts

- What is the difference between recall, discrimination and transfer?
- How does calibration expose false confidence?
- Why can a correct schematic interpretation still support an unsafe conclusion?
- What makes an error log actionable?
