# Block 23 - Main Switches and Control Points

## Core recall
A control point must be mapped to its evidenced scope. Device names and labels indicate intended function but do not prove that every circuit, conductor, auxiliary function or separate source is controlled.

## Retrieval prompts
- What source-to-load path is evidenced?
- Which conductors or circuits does each control affect?
- What exception paths remain possible?
- How can an overbroad whole-installation claim be rewritten safely?
