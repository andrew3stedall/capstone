# Block 62 - Final Cumulative Retrieval

## Recall prompt
Can I select and connect the correct workflows across a mixed case before checking notes?

## Retrieval chain
1. Parse the decision request
2. Retrieve the workflow
3. Separate facts, assumptions and unknowns
4. Map dependencies
5. Write bounded findings
6. Check notes only afterwards
7. Log omissions and overclaims
8. Select one final review priority

Retrieval quality is shown by correct workflow selection and calibrated conclusions, not speed alone.
