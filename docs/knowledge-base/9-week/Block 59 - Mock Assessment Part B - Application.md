# Block 59 - Mock Assessment Part B - Application

## Recall prompt
Can I apply provisional rules to scenario facts while keeping assumptions, dependencies and review boundaries visible?

## Application chain
1. Decision and facts
2. Provisional rule records
3. Assumptions and unknowns
4. Alternatives
5. Dependencies
6. Sensitivity and contradiction checks
7. Provisional selection
8. Limits and review conditions

A bounded recommendation is not a compliant design or field instruction.
