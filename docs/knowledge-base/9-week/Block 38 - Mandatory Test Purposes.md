# Block 38 - Mandatory Test Purposes

## Core idea
Learn each authorised test through the verification question, property, coverage, evidence and limitation it addresses. A result does not prove more than its defined scope.

## Retrieval prompts
- Why is a test name not a complete method?
- What five fields explain a test purposefully?
- Why must the mandatory set be checked against current authorised sources?

## Safety boundary
No test set, method, instrument, connection, value or acceptance criterion is provided. These require authorised sources, approved procedures and competent persons.
