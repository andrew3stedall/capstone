# Block 58 - Mock Assessment Part A - Rule Finding

## Recall prompt
Can I show a traceable path from a decision question to a provisional rule record without inventing authority?

## Rule-finding record
1. Decision question
2. Authorised source category
3. Search vocabulary
4. Candidate provision
5. Applicability checks
6. Confidence status
7. Copyright-safe paraphrase
8. Verification action

A candidate provision is not an applicable rule until scope, definitions, exceptions, edition and jurisdiction are checked.
