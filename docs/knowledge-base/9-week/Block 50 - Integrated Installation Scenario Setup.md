# Block 50 - Integrated Installation Scenario Setup

## Retrieval anchors
- Start with the decision request, not calculations.
- Bound the physical, functional and documentary scope.
- Map normal, alternative and transition states.
- Separate facts, assumptions, contradictions and unknowns.
- Turn missing technical authority into explicit reference questions.

## Scenario-frame checklist
Decision → boundary → purposes → operating states → evidence inventory → assumption register → contradictions → reference questions → stop conditions.

## Safety boundary
The scenario is fictional and documentary. It does not authorise design, switching, isolation, inspection, testing or compliance conclusions.
