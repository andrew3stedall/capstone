# Block 29 — Installation Purpose and Circuit Division

## Core idea
Circuit division should follow installation purpose, loss consequence and operational evidence rather than a copied default layout.

## Retrieval prompts
- What functions must the installation support?
- Which losses would create a common-mode problem?
- What alternative groupings are plausible?
- Which requirements still need authorised verification?

## Evidence habit
Separate observed or stated functions from planning assumptions. Compare at least two grouping strategies before settling on a provisional rationale.

## Safety boundary
This note does not establish compliant circuit arrangements, ratings or mandatory separation. Those decisions require current authorised sources and qualified review.
