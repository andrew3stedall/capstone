# Block 56 - Rest, Reflection and Catch-Up

## Recall prompt
Can I identify one blocking performance pattern, correct it with minimal review, test it on a changed case and stop?

## Recovery loop
1. Recall without notes
2. Select one blocking gap
3. Classify the error
4. Review minimally
5. Create one correction artifact
6. Test a changed case
7. Record recurrence and cue
8. Stop at 25 minutes
