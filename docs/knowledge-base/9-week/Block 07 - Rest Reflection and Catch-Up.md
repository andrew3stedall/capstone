# Block 07 - Rest Reflection and Catch-Up

## Core idea

Use evidence, not guilt, to choose one bounded recovery action. Protect the rest block by defining a stopping condition before catch-up begins.

## Retrieval prompts

- What distinguishes fragile knowledge from unfinished work?
- Which gap genuinely blocks the next week?
- What is the smallest useful recovery action?
- What stopping condition preserves recovery?
