# Block 25 — Source-State Mapping

## Core idea
A source inventory names possible energy origins. A source-state matrix shows which paths are confirmed, excluded, possible or unknown in each defined operating condition.

## Retrieval prompts
- What makes an operating state specific enough to analyse?
- Why must transition states be included?
- What evidence supports an excluded path?
- Which unresolved cells block a safety conclusion?

## Evidence habit
Attach evidence or an evidence request to every non-obvious matrix cell. Do not let mode names, labels or indicator states stand alone as proof.

## Safety boundary
This note does not provide an isolation sequence or verification procedure. Use current authorised sources, equipment information, site procedures and qualified review.
