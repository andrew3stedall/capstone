# Block 17 - Fault-Current Path Reasoning

## Core recall
A valid conceptual fault path must form a complete loop back to the source. A drawn path is not evidence that continuity, impedance or protective performance meets authorised requirements.

## Retrieval prompts
- Where does the path begin and return?
- Which segment contains the fault?
- Which connections are observed, documented, inferred or unknown?
- What separate evidence is required before claiming protective performance?
