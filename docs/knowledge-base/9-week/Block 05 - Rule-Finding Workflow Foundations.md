# Block 05 - Rule-Finding Workflow Foundations

## Core idea

A strong rule-finding process starts with the decision and context, checks definitions and scope, follows cross-references and records a narrow reproducible conclusion.

## Retrieval prompts

- How do object, condition, location and decision improve a search?
- Why should definitions and scope be checked before an apparent requirement?
- What makes an answer traceable?
- How should source-supported facts be separated from interpretation?
