# Block 10 - Current Paths in Normal Operation

## Core idea

Trace the stated circuit condition before deciding whether a complete normal operating path exists. Keep outgoing, intended return and protective paths conceptually separate.

## Retrieval prompts

- What information defines a circuit state?
- Why does a drawn conductor not prove current is flowing?
- Why is the protective path not the normal load-current return?
- What changes when a control point opens?

## Evidence boundary

This note supports conceptual diagram reasoning only. Real isolation, identification and testing require current authorised procedures, suitable equipment, competency and qualified review.
