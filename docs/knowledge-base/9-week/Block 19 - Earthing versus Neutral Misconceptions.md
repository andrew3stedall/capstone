# Block 19 - Earthing versus Neutral Misconceptions

## Core recall
Neutral and protective earthing roles must be traced separately. A relationship at an authorised point does not make the conductors interchangeable throughout an installation, and appearance alone does not verify function or continuity.

## Retrieval prompts
- Which path carries normal load current in the stated condition?
- Which path is intended to support protection under fault conditions?
- What counterexample exposes the claim that neutral and earth are “the same”?
- Which evidence is still needed before field identification?
