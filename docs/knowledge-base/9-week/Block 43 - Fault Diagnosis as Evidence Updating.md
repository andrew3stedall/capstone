# Fault Diagnosis as Evidence Updating

Fault diagnosis should reduce uncertainty rather than reward fast guessing. Keep several hypotheses open, state what each predicts, and update support when evidence arrives.

## Retrieval prompts
- What is the difference between a symptom and a hypothesis?
- How can evidence strengthen one hypothesis while leaving another unresolved?
- Why must disconfirming evidence be actively sought?
- What makes an evidence question discriminating?

## Compact model
**Symptom → competing hypotheses → predictions → bounded evidence → update every hypothesis → residual uncertainty → next authorised question or escalation.**
