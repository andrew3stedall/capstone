# Block 24 - Alternative and Multiple Supplies

## Core recall
A source inventory includes normal, alternative, auxiliary and stored-energy sources. Conclusions must be tied to a defined boundary and operating state; opening the usual control does not address unlisted sources or backfeed paths.

## Retrieval prompts
- What boundary is being analysed?
- Which source categories are present or plausible?
- What changes across normal, transferred, shutdown and faulted states?
- Which evidence request resolves the highest-risk unknown?
