# Block 51 - Planning Evidence Pack

## Retrieval anchors
- Start from the approved scenario frame.
- Compare grouping options before selecting one.
- Keep demand inputs, methods and assumptions separate.
- Treat conductor and protection choices as multi-variable evidence problems.
- End with a bounded recommendation and open technical items.

## Planning-pack chain
Scenario frame → purpose and grouping → demand basis → selection envelopes → coordination questions → option comparison → decision trace → bounded recommendation.

## Safety boundary
This note supports educational planning reasoning only. It supplies no compliant design, component choice, value, limit or installation instruction.
