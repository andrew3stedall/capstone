# Block 53 - Switching and Alternate-Supply Review

## Recall prompt
Can I identify every energy source, define each control's purpose and scope, and reason across stable and transition states without inventing an isolation sequence?

## Review lens
1. Source inventory
2. Control purpose
3. Control scope
4. State matrix
5. Backfeed, bypass and exception paths
6. Evidence contradictions
7. Isolation-evidence gaps
8. Stop conditions and bounded conclusion

An off state, label or control action is not by itself proof that every relevant energy path is isolated.
