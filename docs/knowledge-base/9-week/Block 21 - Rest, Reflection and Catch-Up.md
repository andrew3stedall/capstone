# Block 21 - Rest, Reflection and Catch-Up

## Core recall
A blocking gap prevents safe progress into the next prerequisite concept. A non-blocking gap can be scheduled later. Use at most one correction cycle—recall, compare, correct, re-test and stop—then preserve the recovery boundary.

## Retrieval prompts
- Which Week 3 error, if any, blocks Week 4?
- What evidence distinguishes low confidence from failed recall?
- What is the one highest-impact correction?
- What restart cue will make the next session easy to begin?
