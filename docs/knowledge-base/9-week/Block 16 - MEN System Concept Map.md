# Block 16 - MEN System Concept Map

## Core recall
Separate normal-current conductors from protective conductors. Treat the neutral-earth relationship, its location and its form as authorised-source questions rather than remembered facts.

## Evidence prompts
- What is the conductor's function?
- Is the path normal, protective or unresolved?
- Which current source verifies the relationship?
- What uncertainty requires stopping or escalation?
