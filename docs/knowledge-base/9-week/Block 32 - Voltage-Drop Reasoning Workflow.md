# Block 32 — Voltage-Drop Reasoning Workflow

## Core idea
Voltage-drop conclusions are only as reliable as their assessment boundary, route length, current basis, conductor data, supply context and authorised criterion.

## Retrieval prompts
- What are the start and end points of the assessment?
- Which length and current basis apply?
- Which conductor data and criterion are authorised?
- Which uncertainty most changes the conclusion?

## Evidence habit
Keep every input traceable and unit-controlled. Use sensitivity checks rather than hiding uncertain length, current or allocation assumptions.

## Safety boundary
This note supplies no voltage-drop limits, formula constants or compliant outcomes. Current authorised criteria and qualified review are required.
