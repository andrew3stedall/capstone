# Block 49 - Rest Reflection and Catch-Up

## Retrieval anchors
- Diagnose before reviewing.
- Repair exactly one blocking error pattern.
- Use a changed fictional case to test transfer.
- Record reference-dependent questions instead of guessing.
- Stop at the recovery boundary.

## One-gap recovery cycle
Retrieval → classify → choose one blocker → targeted review → correction artifact → changed-case re-test → carry-forward note → stop.

## Safety boundary
This note supports documentary reasoning only. It does not authorise field diagnosis, testing, isolation, energised work, dismantling or repair.
