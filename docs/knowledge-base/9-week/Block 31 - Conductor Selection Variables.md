# Block 31 — Conductor-Selection Variables

## Core idea
Conductor selection is a constrained evidence problem. Load, installation method, environment, grouping, protection, voltage drop, mechanical exposure and terminations must be considered together.

## Retrieval prompts
- Which variables define the selection envelope?
- Which inputs are verified and which remain assumptions?
- Which variable is most likely to govern the decision?
- Where is current authorised data required?

## Evidence habit
Maintain a selection-variable register. Do not treat design current or a preliminary conductor choice as proof of complete suitability.

## Safety boundary
This note provides no conductor sizes, correction factors or current-carrying values. Final selection requires current authorised sources and qualified review.
