# Block 22 - Functional Switching versus Isolation

## Core recall
Functional switching changes ordinary operation. Isolation is a safety-purpose separation claim that depends on all relevant sources, stored energy, authorised requirements and verification. A stopped or OFF state is not enough.

## Retrieval prompts
- What is the intended purpose of the action?
- Which sources and stored-energy paths remain possible?
- What evidence describes control state, and what evidence establishes source state?
- Which unknown forces a stop-and-verify conclusion?
