# Day 21 - Week 3 Simulated Visual Inspection

## Learning module

- [Day 21 — Week 3 Simulated Visual Inspection](../learning-plans/4-week/modules/day-21-week-3-simulated-visual-inspection.md)
- [[Four-Week Capstone Learning Plan]]

## Core concept

A simulated visual inspection is a bounded evidence exercise. It separates observations, evidence gaps, potential concerns and verified conclusions, then scans supply, switchboard, wiring-system, equipment, special-location and documentation zones.

## Retrieval model

Use **I-N-S-P-E-C-T**: Identify scope; Note sources and functions; Scan by zone; Pinpoint observations; Evaluate evidence; Classify provisionally; Transfer the record.

## Assessment evidence

The learner must define scope, complete the six-zone scan, write traceable finding records, re-grade claims after changed evidence and deliver a bounded handover. Invented causes, official defect codes or field instructions are critical errors.

## Safety and review boundary

No approaching, opening, touching, operating, isolation, testing, repair or alteration is authorised. Formal inspection scope, required items, response classifications, reporting and acceptance criteria remain `reference_check_required` and not `technically-reviewed`.
