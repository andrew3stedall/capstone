# Six-Week Day 03 — Fundamental Protection Concepts and Fault Types

## Purpose

This note connects safety-pathway analysis to later protection selection. The learner separates symptom, path, mechanism, consequence and protection purpose before naming a protective device.

## Core model

Use **F-A-U-L-T**:

- **Fix the facts** — record the supplied circuit arrangement, sources and symptoms; grade the evidence.
- **Analyse the path** — trace intended current flow and any abnormal path or discontinuity.
- **Understand the mechanism** — distinguish overload, short circuit, earth fault, leakage or residual-current condition, open circuit and unresolved cases.
- **Link the protection purpose** — state whether the immediate concern is heating, shock exposure, fault energy, equipment damage, continuity or another consequence.
- **Test the evidence boundary** — grade the claim and identify the authorised source and technical data needed before selection, testing or practical action.

The reusable reasoning chain is:

**symptom → initiating condition → current path → mechanism → consequence → protection purpose → evidence boundary**

## Evidence and claim grades

### Evidence grades

- **Observed:** directly stated or visible in the supplied scenario.
- **Documented:** supported by a current drawing, schedule, label, record or authorised document.
- **Derived:** calculated or logically inferred from verified inputs using an applicable method.
- **Assumed:** plausible but not evidenced.
- **Missing:** required for the conclusion but unavailable.

### Claim grades

- **Described:** states only what the supplied evidence shows.
- **Supported:** links applicable evidence to a bounded classification.
- **Provisional:** plausible, but one or more material facts remain unverified.
- **Unresolved:** available evidence does not distinguish safely between credible mechanisms.

## Practical application

Classify fictional cards containing:

1. increased demand in an otherwise intact circuit;
2. unintended connection between different potentials;
3. an unintended earth-associated path;
4. current imbalance with unknown cause;
5. discontinuity in an intended path;
6. incomplete evidence that requires provisional or unresolved classification.

For each, state the known facts, intended path, abnormal path or discontinuity, evidence grades, fault category, claim grade, possible consequence, protection purpose, missing evidence, reopening trigger and stop boundary.

The key discipline is:

> Protective-device operation is an observation. It is not proof of fault type, location or cause.

## Assessment relevance

A strong response:

- distinguishes symptom, mechanism and consequence;
- maps intended and abnormal current paths;
- separates overload from short-circuit reasoning;
- treats residual current as evidence of imbalance rather than a complete diagnosis;
- grades evidence and claims consistently;
- refuses unsupported device, test or practical-action conclusions;
- reopens the classification when a source, path or material fact changes.

The module uses a six-category, 12-point educational rubric. A critical error overrides the numerical result. This is not an official assessment rule.
