# 12-Week Day 31 - Fault-Loop Reasoning at Concept Level

## Purpose

Builds a complete source-to-fault-and-return mental model without substituting conceptual reasoning for authorised calculation, testing or technical approval.

## Core concepts

Fault loop; loop continuity; loop impedance; prospective fault current; automatic disconnection; bottleneck; described, supported and verified claims.

## L-O-O-P-S workflow

Locate the source and fault; outline outgoing and return segments; observe evidence quality; predict consequences; source-check the criterion and bound the conclusion.

## Practical application and assessment relevance

Learners annotate fictional loops, identify missing evidence, predict directional effects, reopen affected design records and complete a faded independent scenario. Assessment focuses on path completeness, terminology, evidence grading, consequence reasoning, source discipline and claim restraint.
