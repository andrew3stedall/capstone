# Six-Week Day 18 - Voltage-Drop Concepts and Calculation Workflow

## Purpose

Build an evidence-controlled voltage-drop calculation and return the result to the complete circuit-design chain.

## Core workflow

**V-O-L-T-A-G-E:** Verify boundary → Organise evidence → Locate authorised method → Translate units → Apply transparently → Gauge against criterion → Extend into the design chain.

## Practical application

Prepare an evidence ledger, calculate only from fictional or authorised training data, and identify every recalculation trigger.

## Assessment relevance

The learner must define the circuit boundary, use correct units, trace each input to evidence, avoid remembered limits and explain downstream reopening.
