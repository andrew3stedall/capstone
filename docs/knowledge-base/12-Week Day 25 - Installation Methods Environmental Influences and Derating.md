# 12-Week Day 25 - Installation Methods Environmental Influences and Derating

## Purpose

Supports route segmentation, evidence-based installation classification and single application of verified influences.

## Core concepts

- **Route section:** a portion with materially consistent conditions.
- **Environmental influence:** a surrounding condition relevant to selection or capacity.
- **Correction factor:** an authorised adjustment tied to a defined condition.
- **Governing section:** the controlling section for a particular check.
- **Double application:** counting the same influence more than once.

## C-O-N-D-I-T-I-O-N-S workflow

Capture the route; outline sections; name observed facts; determine authorised classifications; identify influences; trace source values; implement each factor once; order results; note unresolved dependencies; state reopening triggers.

## Practical application

Learners build route-section registers, audit factor provenance and transfer changed conditions without guessing.

## Assessment relevance

Observable evidence includes segmentation, source tracing, factor control, governing-case reasoning, uncertainty handling and bounded conclusions.
