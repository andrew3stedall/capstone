# Day 11 - Voltage Drop

Day 11 develops an evidence-based voltage-drop workflow. The learner defines the operating case and calculation endpoints, reconstructs every contributing section, obtains authorised conductor data and calculation conventions, combines section results and records a bounded conclusion.

## Core evidence model

A defensible voltage-drop result connects:

1. **duty and operating case** — steady running, starting, intermittent or another defined condition;
2. **source and destination** — the exact points between which voltage performance is assessed;
3. **complete path** — consumer mains, submains, final subcircuits and any other contributing sections;
4. **current basis** — the current applicable to each section and case;
5. **conductor evidence** — material, size, arrangement, temperature assumptions and authorised data;
6. **calculation convention** — source-controlled length, phase, neutral, resistance, reactance and power-factor treatment;
7. **cumulative result** — section contributions combined using the authorised method;
8. **bounded conclusion** — equipment needs, compliance requirement, assumptions and unresolved evidence.

The module uses the **D-R-O-P** workflow: define duty and destination, reconstruct the route, obtain authorised method and inputs, then prove the cumulative result and iterate.

## Practical application

Use the workshop compressor and lighting-extension scenario in the linked module. Build separate starting and running cases, map the complete path, prepare a source-controlled worksheet and compare design responses without inserting real standards constants.

## Assessment relevance

The block samples whether the learner can:

- distinguish voltage drop from supply variation and current-carrying capacity;
- identify all contributing circuit sections;
- select an operating current only after stating its basis;
- distinguish physical route length from calculation length;
- keep conductor data and formula conventions tied to authorised sources;
- consider steady and transient equipment cases;
- propose design iterations without overlooking upstream limitations;
- state why a provisional result is not yet a compliance claim.

## Misconceptions to track

- Only the final subcircuit contributes to voltage drop.
- Protective-device rating is always the correct calculation current.
- One-way physical length can be inserted into any remembered formula.
- A thermally adequate conductor automatically passes voltage drop.
- A design budget is the same as an official limit.
- Motor starting can be assessed from the steady running case alone.
- A larger final-subcircuit conductor fixes every upstream limitation.
- A calculated low voltage proves that voltage drop, rather than a fault, is the cause.
