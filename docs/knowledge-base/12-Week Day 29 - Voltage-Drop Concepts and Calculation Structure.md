# 12-Week Day 29 - Voltage-Drop Concepts and Calculation Structure

## Purpose

Builds a transparent voltage-drop calculation structure while keeping calculation, interpretation and compliance claims separate.

## Core concepts

- **Calculation boundary:** the defined start and end points.
- **Path convention:** how the selected method represents the electrical path.
- **Conductor data:** sourced resistance, reactance, impedance or tabulated information.
- **Upstream contribution:** voltage drop before the present boundary.
- **Dimensional check:** confirmation that units produce the intended result unit.

## V-O-L-T-S workflow

Verify the boundary and operating case; obtain the authorised method and data; lock the path convention; transform visibly; state the bounded result.

## Practical application

Learners build an input register, audit units, complete a fictional visible calculation, compare boundaries and recalculate after a changed condition.

## Assessment relevance

Observable evidence includes boundary definition, provenance, path control, unit discipline, transparent transformation and claim restraint.
