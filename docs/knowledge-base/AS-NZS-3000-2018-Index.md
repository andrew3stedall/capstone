## Source record

- **Title:** AS/NZS 3000:2018 Electrical installations (Australian/New Zealand Wiring Rules)
- **Edition:** 2018, including any applicable amendments to be verified
- **Publisher:** Standards Australia / Standards New Zealand
- **Repository status:** reference metadata only; the copyrighted PDF is not committed

> This index is not a substitute for the Standard. Clause details, exact requirements, values, tables and figures must be checked against an authorised current copy.

## How to use this index

Each topic below should become one or more original learning notes. A note should explain the concept in learner-focused language, use independently created examples and scenarios, and reference the official clause rather than reproducing it.

## Coverage workflow

For every section:

1. verify headings and current amendment status;
2. create a clause-to-concept map without copying clause text;
3. group clauses into learner-centred modules;
4. define prerequisites and backlinks;
5. draft original explanation and diagrams;
6. add retrieval questions and scenarios;
7. add practical/simulator links;
8. conduct qualified technical review;
9. record source edition, clause references and review date;
10. publish only after copyright and technical checks.

## Current limitations
