# Six-Week Day 17 - Installation Conditions and Derating-Factor Reasoning

## Purpose

Classify route and thermal conditions before applying any authorised correction method, then return the adjusted capacity to the full design chain.

## Core workflow

**C-O-N-D-I-T-I-O-N:** Capture route → Observe evidence → Name influences → Determine reference condition → Identify corrections → Test interactions → Integrate adjusted capacity → Observe consequences → Note the bounded conclusion.

## Practical application

Map a fictional route section by section, classify evidence, audit supplied training factors and reopen the design when grouping, ambient temperature or insulation changes.

## Assessment relevance

The learner must separate base and adjusted capacity, justify every condition and factor, prevent double correction and explain which downstream decisions reopen.
