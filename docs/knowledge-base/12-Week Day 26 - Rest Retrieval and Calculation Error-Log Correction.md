# 12-Week Day 26 - Rest Retrieval and Calculation Error-Log Correction

## Purpose

Provides a bounded recovery block for spaced retrieval, error diagnosis and correction without adding new electrical theory.

## Core concepts

- **Error log:** record of error, cause, repair, affected work and prevention cue.
- **Error categories:** arithmetic, unit, evidence, method, transcription and conclusion.
- **Downstream reopening:** repeating later work affected by an upstream correction.
- **Recovery limit:** maximum 30 minutes and no more than three repairs.

## R-E-P-A-I-R workflow

Retrieve first; expose the error; pinpoint the category; amend one cause; identify downstream effects; record prevention and rest.

## Practical application

Learners retrieve four recent workflows, triage up to three errors, make one visible repair per error and issue a bounded readiness statement.

## Assessment relevance

Observable evidence includes accurate classification, transparent repair, downstream reopening, uncertainty control and fatigue-aware stopping.
