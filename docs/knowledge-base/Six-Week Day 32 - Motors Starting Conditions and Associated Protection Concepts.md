# Six-Week Day 32 - Motors Starting Conditions and Associated Protection Concepts
## Purpose
Separate motor operating states, power and control paths, and protection purposes before making any equipment decision.
## Core workflow
**M-O-T-O-R-S:** Map task → Obtain data → Trace paths → Outline protection purposes → Review starting/restart evidence → State bounded result.
