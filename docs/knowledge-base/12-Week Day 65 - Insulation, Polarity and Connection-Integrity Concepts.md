# 12-Week Day 65 - Insulation, Polarity and Connection-Integrity Concepts

## Purpose

A document-based method for separating insulation, polarity and connection-integrity evidence so conclusions do not exceed the question, boundary, state or provenance of the source record.

## Core model

Use **S-E-P-A-R-A-T-E**: State the evidence question; Establish the boundary; Preserve the result; Attach state and provenance; Review contradictions; Avoid false transfer; Test completeness; Express a bounded conclusion.

## Practical application

Prepare a separated evidence matrix that records the purpose, circuit boundary, state, exclusions, source observation, supported interpretation, unsupported transfer and reopening trigger for each evidence type.

## Assessment relevance

A strong response keeps continuity, insulation, polarity and connection-integrity claims distinct, records changed installation state and does not invent procedures, values or compliance conclusions.
