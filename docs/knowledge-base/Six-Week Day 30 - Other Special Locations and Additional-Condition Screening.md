# Six-Week Day 30 - Other Special Locations and Additional-Condition Screening

## Purpose

Build a traceable screening conclusion by separating observed conditions from candidate specialist classifications, verifying applicability through current authorised sources and reopening every dependent decision.

## Core workflow

**S-C-R-E-E-N:** State boundary → Capture observed features → Rank trigger families → Establish source queries → Extract applicability and reopen dependencies → Note grades, gaps, triggers and bounded action.

## Evidence and claim model

- Evidence grades: **recalled**, **located**, **supported**, **transferred**, **unresolved**.
- Claim grades: **memory claim**, **provisional interpretation**, **supported study conclusion**, **authorised technical determination**.
- Applicability ledger: boundary → observations → trigger family → source query → missing evidence → dependent decisions → evidence grade → claim grade → reopening triggers → bounded action.
- A room name, marketing term, product label, photograph or remembered example is not applicability proof.

## Practical application

Complete applicability ledgers for farm, temporary-work, pool-plant and commercial-kitchen scenarios. Separate observations from candidate trigger families, draft current-authorised-source queries, identify dependent decisions, grade evidence and claims, perform one changed-condition transfer and state a bounded action.

## Assessment relevance

Tests observable applicability reasoning: boundary control, observation/inference separation, trigger screening, source navigation, dependency reopening, changed-condition transfer and bounded reporting. The 12-point rubric and critical-error gates are original educational tools, not an official RTO pass mark.

## Reopening triggers

Reopen a conclusion when location use, occupancy, activity, cleaning method, environmental exposure, equipment identity or duty, source arrangement, access control, physical boundary, jurisdiction or source currency changes or conflicts with later evidence.
