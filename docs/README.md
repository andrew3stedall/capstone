# Capstone Ready Documentation

This directory is the canonical design, learning, content and implementation reference for **Capstone Ready**.

The application is an independent study companion for Australian electrical apprentices preparing for capstone assessment. It is not an official assessment provider and does not replace current legislation, regulator guidance, RTO procedures or licensed copies of applicable standards.

## Product pillars

1. **Understand** — concise explanations supported by a deep reference library.
2. **Remember** — highlights, notes, flashcards, retrieval practice and spaced review.
3. **Apply** — scenarios, calculations, wiring exercises and testing workflows.
4. **Diagnose** — fault-finding exercises driven by realistic evidence.
5. **Rehearse** — practical and theory assessment modes with feedback.

## Documentation rules

- Original learning content must be written in plain language.
- Standards may be indexed and referenced, but copyrighted text, tables and figures must not be systematically reproduced without permission.
- Exact pass marks, assessment structure and procedures must be labelled by jurisdiction and RTO where they vary.
- Safety-critical guidance must identify its source and review date.
- Every learning module should link to related concepts, practice activities, faults, tests and references.
