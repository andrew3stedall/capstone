# Milestone 3 technical review checklist

## Review gate

A suitably qualified Australian electrical reviewer must complete this checklist before any Milestone 3 module changes from `review-required` to `technically-reviewed`.

## Modules

- [ ] Safety and electrical risk
- [ ] Electrical fundamentals
- [ ] Control, switching and protection concepts
- [ ] Verification overview

## For each module

- [ ] Confirm the explanation is technically accurate and appropriate for apprentices.
- [ ] Confirm no statement could encourage unsafe work or bypass supervision.
- [ ] Check terminology against current authorised sources.
- [ ] Check jurisdiction-sensitive claims and add limits where needed.
- [ ] Verify calculations and original examples independently.
- [ ] Confirm source edition, currency date and links.
- [ ] Confirm the module does not reproduce protected standards wording, tables or figures.
- [ ] Record reviewer name, qualification, review date and requested changes.
