# Capstone Ready

Capstone Ready is a modern study and practice app for Australian electrical apprentices preparing for their final capstone assessment.

## What is included

- Readiness dashboard with weak-area prioritisation and a daily study plan
- 11 learning paths mapped across the 48 ERAC Essential Performance Capabilities (2024)
- Interactive practice questions with answer reasoning and critical-evidence labels
- Timed mock-exam modes and a practical-workflow rehearsal checklist
- Ohm's law, power and balanced three-phase calculation tools
- Progress, mastery, activity and coverage views
- Australian jurisdiction selector and standards/regulator reminders
- Responsive desktop, tablet and mobile layouts

## Coverage model

The learning paths cover:

1. Safety and electrical risk
2. Electrical fundamentals
3. Wiring rules and design
4. Energy systems, renewables, storage and EV charging
5. Conductors and circuits
6. Switchgear and control gear
7. Accessories and equipment
8. Earthing and MEN
9. Motors and transformers
10. Isolation, testing, faults and commissioning
11. Hazardous, special and high-voltage awareness areas

The current question bank is original demonstration content, not copied exam material. It intentionally avoids reproducing standards tables or presenting the app as an official assessment provider.

## Run locally

```bash
npm ci
npm run dev
```

Quality checks:

```bash
npm run lint
npm run test
```

## Important limitation

This is an independent learning app. Electrical work and assessment answers must be checked against current legislation, standards, local regulator guidance and the learner's RTO-approved procedures.
